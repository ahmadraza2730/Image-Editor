# 🖼️ Image Editor (C++)

A C++ application for performing core image editing operations — built from the ground up to understand image processing fundamentals at the pixel level.

## 📖 About

This project implements a functional image editor in C++, focusing on manipulating raw image data directly. It was built to strengthen understanding of file I/O, pixel-level data structures, and algorithmic image transformations without relying on high-level editing libraries.

## ✨ Features

- 🎨 Basic filters (grayscale, invert, blur, sharpen)
- 🔆 Brightness & contrast adjustment
- 🔄 Image rotation & flipping
- ✂️ Cropping and resizing
- 💾 Load and save support for common image formats
- 🧮 Pixel-level manipulation using custom logic

## 🛠️ Tech Stack

- **Language:** C++
- **Libraries:** stb_image / stb_image_write *(update if you used a different image library, e.g. OpenCV)*
- **Concepts used:** File I/O, Arrays/Matrices, Pixel Manipulation, Algorithms

## 🚀 Getting Started

### Prerequisites
- A C++ compiler (g++ recommended)

### Run Locally
```bash
git clone https://github.com/ahmadraza2730/Image-Editor.git
cd Image-Editor
g++ main.cpp -o ImageEditor
./ImageEditor
```
